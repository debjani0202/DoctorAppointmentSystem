# Doctor Appointment Booking System

A console-based Java application built using JDBC and MySQL that allows patients to book appointments with doctors.

## Features
- Patient registration and login
- Doctor management
- Slot creation and availability tracking
- Appointment booking using available slots
- DAO-based architecture

## Technologies Used
- Java
- JDBC
- MySQL
- Git & GitHub

## How to Run
1. Create the database and tables using MySQL.
2. Update database credentials in `DBConnection.java`.
3. Run test classes from the `main` package.
